SSH

SSH是一个协议，用来实现计算机之间的加密登录
SSH使用的是非对称性加密算法（了解）

如何在bash下通过ssh协议进行远程服务器的管理

ssh 命令是OpenSSH提供的，windows下没有这个命令，如果要在windows下使用ssh，需要安装第三方的工具（putty）

ssh如何实现免密登录

  1. 首先使用如下命令在本机生成公钥以及私钥

  2. 使用用户名和密码通过ssh先登录到服务器

  3. 找到服务器中用户根目录下的.ssh文件夹中的auterized_keys文件 (~/.ssh/autherized_keys)
  4. 将本机用户根目录下面的.ssh文件夹中的id_rsa.pub文件中的内容复制粘贴到上一步中找到的文件中
  5. 退出服务器，再次使用ssh命令登录服务器的时候，就不需要密码了

github

源代码托管平台（远程仓库服务器）
操作github的方式其实就是我们讲过的操作远程仓库的方式！

配置github免密

  1. 打开github
  2. 点击头像，选择下拉框中的settings
  3. 点击settings页面左侧的SSH keys And GPG Keys
  4. 点击new SSH key
  5. 找到自己电脑中 用户根目录下面.ssh文件夹中的 id_rsa.puh文件(这一步操作需要之前已经执行过 ssh-keygen -t rsa)
  6. 将该文件中的内容完全复制到 刚才的页面中的key文本框中
  7. title 随便写，但一般使用key中粘贴的内容的最后一句
  8. 点击add ssh key

tips: 只有通过ssh连接才能够实现免密

github静态页面托管

  1. 只能有一个这样的项目
  2. 项目的名称为特定的格式 github用户名.github.io
  3. 除了第2步中的方式，还可以将任意的项目作为静态页面来展示，但是遵守第一条原则
    3.1 随便找一个自己的项目
    3.2 找到项目的settings
    3.3 在settings页面中找到github pages的配置
        3.4 选择一个要作为页面源代码的分支
            3.5 点击save按钮，就可以了
